This project investigates the systematic comparison of agentic platforms for software engineering, focusing on tools that integrate autonomous or semi-autonomous agents into the software development lifecycle. The study develops a structured evaluation framework to compare multiple agentic platforms across dimensions such as functionality, usability, collaboration support, scalability, and reliability. The DESMET systematic methodology will be applied for this study. The study requires to select, try, and systematically evaluate the different available platforms.

Step 0: define a timeline for the rest of the study. 

Step 1: define a set of sources for the initial search of agentic platforms: search through Google, Reddit, specialised forums, X, LinkedIn, papers using well-known platforms, etc. (if you do not find papers because they are not accessible, you ask Alessio).
Step 2: define a set of criteria for selection of the most mature or promising platforms (around 10).
Step 3: Apply search an selection criteria to identify the platforms that will be used.
Step 4: Define the analysis criteria, following DESMET guidelines (reference here, cf. example here). To define the criteria you can interact with people who are using or willing to use these platforms (e.g., personal contacts, forums, linkedin, etc.). 
Step 5: Define one or more problems that we can use to exercise the tools.
